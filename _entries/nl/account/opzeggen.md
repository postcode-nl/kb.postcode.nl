---
title: Hoe kan ik mijn API abonnement opzeggen?
---
Opzeggingen voor de API kunnen gedaan worden door in te loggen op de [beheeromgeving](https://account.postcode.eu/home/nl_NL), dit moet door de admin van het account gebeuren.

In het menu 'Beheer' staat een item Abonnement. Hier kan het abonnement dan per eerstvolgende mogelijkheid opgezegd worden.
