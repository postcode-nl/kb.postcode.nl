## **Veelvoorkomende problemen**

Ervaar je API-fouten terwijl de API wel online is? Hieronder staan enkele veelvoorkomende problemen die je kunt tegenkomen.

*   Controleer je accountstatus door in te loggen op [account.postcode.eu](https://account.postcode.eu/home/nl_NL). Zorg ervoor dat er geen achterstallige facturen zijn. Facturen die onbetaald blijven, leiden uiteindelijk tot een tijdelijk geblokkeerd account.

*   Heb je een testaccount? Mogelijk zijn je testaanvragen opgebruikt. Log in op [account.postcode.eu](https://account.postcode.eu/home/nl_NL) om je accountstatus te controleren en een abonnement aan te vragen.

*   Als je een plugin van een derde partij gebruikt om toegang te krijgen tot onze API, controleer dan of de licentie nog geldig is. Neem contact op met je plugin-leverancier om dit te controleren.
