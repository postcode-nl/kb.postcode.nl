---
title: Hoe kan ik de API testen?
---
Als je een [account registreert](https://account.postcode.eu/home/nl_NL) dan staat deze altijd eerst in test toegang, je gaat dus nog geen abonnement aan. Je hebt zes maanden de tijd om de API te testen,  en er zit een maximum aan het aantal validaties dat je kan doen in test toegang. Na het testen geef je zelf aan hiermee klaar te zijn en dat het abonnement in mag gaan, daarbij kies je welk abonnement je wilt hebben. 

## Moet ik credentials aanpassen?

Alle gegevens blijven hetzelfde, je hoeft dus niks te wijzigen qua credentials als je site of systeem in productie wordt genomen.

Als je geen account wil registreren maar alleen de werking wil zien, dan is er ook een [demo op de site](https://www.postcode.eu/nl/products/address-api).
