---
title: Hoe worden voor het abonnement de hits op de API geteld?
---
We hebben onze pricing voor de autocomplete International Address API expres zo gemaakt dat alleen volledige lookups worden gefactureerd. Dit betekent dat alle hits die je doet na een toetsaanslag, inclusief de uiteindelijke getDetails call (wanneer het resultaat gevonden is), als 1 lookup wordt geteld.

Voor de simpelere Nederlandse Postcode API wordt wel per hit geteld.
