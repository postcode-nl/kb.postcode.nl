---
title: Wat is het verschil tussen de Autocomplete of Validate API vs. Nederlandse Postcode API?
---
Met de Autocomplete of Validate API kan je adressen opvragen van alle beschikbare landen. Deze API's zijn uitgebreider dan de Nederlandse Postcode API qua functionaliteit. Zo kan je Nederlandse adressen niet alleen aanvullen op basis van een postcode en huisnummer, maar kan je ze ook aanvullen met behulp van de auto complete functionaliteit. Handig als er adressen worden ingevoerd waar de postcode niet van bekend is. De Nederlandse Postcode API werkt met alleen een postcode en huisnummer om een volledig adres terug te krijgen.

**Kan ik maar 1 API gebruiken?**

Je kan uiteraard alle API's naast elkaar gebruiken. Zo kan er voor je internationale klanten adressen worden opgehaald via de Autocomplete of Validate API, en voor Nederlandse klanten via de Nederlandse Postcode API. Bij sommige plugins wordt dit ook al automatisch gedaan, die kiest de juiste API voor als er een adres opgevraagd wordt.Â 

**Waarom is de Nederlandse Postcode API goedkoper?**

Nederlandse adressen die via de Autocomplete of Validate API opgehaald worden hebben een hogere stukprijs omdat deze API's meer functionaliteit biedt. Ook als er een Nederlandse postcode en huisnummer opvraging wordt gedaan op de Autocomplete of Validate API geldt dat tarief.
