---
title: Welke plugins zijn er beschikbaar?
---
Alle bij ons bekende plugins hebben wij op onze [implementatie pagina](https://www.postcode.eu/nl/products/address-api/implementation) gezet. Zo zijn er o.a. plugins voor Shopware, Magento en WooCommerce.

**Waarom moet ik voor sommige plugins extra betalen?**

De gratis plugins worden 'as is' ter beschikking gesteld, dus zonder support. Aangezien er voor de verschillende platformen veel verschillende plugins zijn, kan het een uitdaging zijn om alles met elkaar te laten samenwerken. De broncode van sommige gratis plugins is daarom vrij toegankelijk zodat de implementatie-partners zo veel mogelijk geholpen worden om de site te bouwen op de manier dat ze zelf willen.

De betaalde versies worden door derde partijen aangeboden die support op hun plugins leveren. Je hebt dan bij twee verschillende partijen een overeenkomst. Bij Postcode.nl voor de API en bij de derde partij voor de plugin. Deze overeenkomsten staan los van elkaar.

**Staat je plugin er niet bij?**

Geen nood, de API kan ook ingebouwd worden met behulp van de [technische documentatie](https://developer.postcode.eu/documentation).
